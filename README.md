Perintah Untuk menjalankan script Virus Lawas

Beri izin dulu:
Bash
chmod +x "VIRUS lama.sh"
Lalu jalankan:
Bash
bash "VIRUS lama.sh"
atau kalau namanya tanpa spasi:
Bash
bash VIRUS_lama.sh
Kalau sudah ada shebang:
Bash
#!/data/data/com.termux/files/usr/bin/bash
dan sudah diberi izin execute:
Bash
chmod +x VIRUS_lama.sh
bisa langsung:
Bash
./VIRUS_lama.sh
Untuk menu Info V1.1, kamu bisa tulis:
Plain text
